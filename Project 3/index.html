Face Generator : The Interactive Fire Particle

Interactive Fire Particle reacts with the MousePressed events

Design Process

The original idea is making a fire particle that moves along with the mousepress direction. and since the project is 'face generator', I decided to put eyes on the fire so that it can be look like a face. and than I designed my initial sketch that can be seen on 'Part1_Sketch.png'file, which is the sketch of the fire moving his eyes and his particle along the mousepress position.

The most struggled part to generate the sketch into the code was making the eyes move along the mouse direction, but I got some help from my classmate and I figured out that using constraint function would be the solution. so I edited my code applying constraint function I succeeded! but what want was making the eyes move to both mouse x and y direction but I only succeeded for x direction
