Optical illusion

Optical illusion Sketch using nested loop function

Design Process

When creating the optical illusion, I wanted to utilize a pattern based on the nested loop code I had created during a class activity. So, I explored ways to create the optical illusion using that code. I decided to create an illusion where all the surrounding stars rotate and expand, while the central star only expands and remains stationary. Because of this contrast in movement, I wanted to induce an illusion where the stationary central star appears to tremble or move unsteadily.
