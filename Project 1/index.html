Draft Data Portrait Design Process
1. Project Motivation and Data Selection
This project aimed to visualize personal data titled 'My Fashion Preferences and Color Usage Habits'. I believed that the consistency and complexity of color choices (qualitative data) better represented my style than simply the number of clothes worn (quantitative data).

Therefore, based on photos of outfits worn daily over 4 days, I conceived the idea of extracting the key colors used in those outfits, measuring the similarity between colors, and creating a visual relationship map.

2. Color Extraction and Data Structuring Process
The core of the data was the color values themselves.

Color Extraction: I extracted 3 to 5 most prominent colors from each outfit photo over the four days. These colors were stored as HSB (Hue, Saturation, Brightness) values, not RGB values.

Reason for Choosing HSB: HSB allows for an intuitive grasp of similarity between colors (Hue). We determined it was the most suitable for this project, which involves placing shapes of visually similar colors close together.

Data Arrangement: All extracted color values were stored in a nested array named fashionData. Each element contains hue, saturation, brightness values, and the proportion it occupies on the garment (size), e.g., {h: 15, s: 75, b: 95, size: 40}.

The code was implemented by integrating all color data from four days onto a single canvas to create a 'Color Relationship Map'.

Array Iteration: We used nested for loops to retrieve every date from the fashionData array and all the color information contained within it, ensuring nothing was missed.

Coordinate Mapping:

X-axis (Hue): I mapped each color's Hue value proportionally to its position on the canvas's X-axis. This causes shapes with similar hues (e.g., shades of yellow) to cluster closely together along the X-axis.

Y-axis (Saturation): Each color's Saturation value was mapped to its Y-axis position on the canvas. High-saturation (vibrant) colors are placed on one side, while low-saturation (near-monochromatic) colors are positioned on the opposite side, revealing patterns in color vividness.

Shape Size: The proportion (size) of each color in the outfit was mapped to the shape's size (radius). Larger circles represent colors of significant items like pants or tops, while smaller circles represent colors of minor accessories, indicating their importance.
