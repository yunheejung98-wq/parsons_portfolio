Abstract Clock

Experimental clock that challenges the conventional, linear notion of time.

Design Process

This is an experimental clock that challenges the conventional, linear notion of time. I designed a one-minute-only clock where 60 lines, which appear to be interconnected, are scattered randomly across a canvas. As each second passes, one line disappears. After 60 seconds have elapsed, the canvas is left completely blank, at which point a new minute immediately begins and a fresh set of lines appears.

The most challenging parts of this clock's creation were making the 60 lines appear interconnected via the squares at their ends and implementing the effect of lines disappearing one by one every second.

With help from Gemini, I implemented the disappearing lines effect using the floor() function. Instead of removing lines one per second, the code redraws them in every millisecond. The number of lines to draw is determined by the equation: 60 (total lines) - floor[(elapsed milliseconds) / 1000]. This ensures that an integer number of lines is redrawn on the canvas each second, creating the visual effect of a line disappearing once per second.

Additionally, to give the clock a unique visual edge, I used an array to randomly assign the color and size of the connecting squares from within a predefined color palette.
